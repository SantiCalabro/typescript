"@types/node": "^24.6.0" y  "@types/express": "^5.0.3" son para poder tipar adentro de express
    npm i -D @types/express
    npm i -D @types/node

    npm i -D morgan ---> Para usar middlewares creo que era esto
    npm i -D ts-node ---> Permite ejecutar archivos .ts directamente en node, sin necesidad de compilar previamente. (Node no ejecuta TS por default. Nos la tendríamos que pasar compilando)
    npm i -D nodemon ---> watcher
    npm i dotenv ----> Para la gestión de las variables de entorno
